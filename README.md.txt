# File Organizer Python

## Description
This project is a Python automation script that organizes files into folders based on their file type such as images, documents, videos, and zip files.

## Features
- Automatically organizes files
- Creates folders if not exist
- Supports multiple file types
- Easy to use

## Technologies Used
- Python
- OS Module
- Shutil Module

## How to Run
1. Install Python
2. Update folder path in file_organizer.py
3. Run the script:
   python file_organizer.py
