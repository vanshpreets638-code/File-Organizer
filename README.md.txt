# File Organizer Python

## Description
This is a Python automation script that organizes files into folders based on file type such as images, documents, videos, and others.

## Features
- Automatically organizes files
- Creates folders by file type
- Moves files into correct folders
- Simple and easy to use

## Technologies Used
- Python
- OS Module
- Shutil Module

## How to Run
python file_organizer.py
